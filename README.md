# PDF分割ツール ・7日試用版

複数ページのPDFファイルを1ページずつの個別ファイルに分割するツールの7日間試用版です。

## 概要

- 試用期間：初回起動から7日間（フル機能）
- 試用期間終了後は有料版をご購入ください
- 有料版：https://apps.microsoft.com/detail/9pm38hpwfngj

## 動作環境

- Windows 10 / 11（64bit）

## インストールについて

本アプリはスタンドアロン実行ファイルです。インストーラーではありません。
ダウンロード後、EXEファイルを直接実行してください。

## 終了コード

- 0: 正常終了

# LTL PDF Splitter

Split multi-page PDF files quickly and easily.

No ads. No subscription. No account needed. Does one thing and does it well.

## Features

- Visual thumbnail preview of all pages
- Adjustable thumbnail size and column layout
- Select multiple split points at once
- One-click batch splitting
- Drag and drop PDF loading
- Runs completely offline — no data is ever sent anywhere
- Lightweight and fast

## How to Use

1. Click 'Open PDF' to select a file
2. Click on pages where you want to split (multiple selections OK)
3. Click 'Split' to execute
4. To split another file, just click 'Open PDF' again

Your original PDF is never modified. Split files are saved to a new folder.

## System Requirements

- Windows 10 / 11 (64-bit)

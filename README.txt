PDF証拠整理ツール
README
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

この度はPDF証拠整理ツールをご購入いただき、誠にありがとうございます。

本ツールは、訴訟・紛争案件における証拠整理業務を効率化するために
開発された専用ツールです。


■ 同梱ファイル
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

□ PDF分割ツール.exe
  - PDFファイルを複数の分割ポイントで一度に分割

□ 証拠番号付与ツール.exe
  - PDFファイルに証拠番号（甲第○号証等）を自動付与

□ 操作説明書.pdf（または.txt）
  - 各ツールの詳しい使い方とFAQ

□ README.txt（本ファイル）


■ 動作環境
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

OS：Windows 11（64bit）
メモリ：8GB以上推奨
ストレージ：500MB以上の空き容量


■ インストール不要
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

本ツールはexeファイル単体で動作します。
特別なインストール作業は不要です。

【使い方】
1. exeファイルをお好きな場所に配置
   （デスクトップやドキュメントフォルダなど）
2. ダブルクリックで起動

※初回起動時、Windowsセキュリティの警告が表示される場合があります。
  「詳細情報」→「実行」をクリックしてください。


■ 初めてお使いになる方へ
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

まずは「操作説明書.pdf」をご覧ください。
画面キャプチャ付きで詳しく解説しています。

【クイックスタート】

＜PDF分割ツール＞
1. exeファイルを起動
2. 「PDFを開く」で対象ファイルを選択
3. 分割したい先頭ページをクリック（複数選択可）
4. 「分割実行」をクリック

＜証拠番号付与ツール＞
1. exeファイルを起動
2. 「フォルダを開く」でPDFファイルを読み込み
3. ファイルの順番を調整（ドラッグ&ドロップ）
4. 証拠種別（甲/乙等）とフォント設定
5. 「証拠番号を付与して保存」をクリック


■ よくある質問
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

Q. 元のPDFファイルが変更されることはありますか？
A. ありません。常に新しいファイルとして保存されます。

Q. Googleドライブに保存できますか？
A. Googleドライブデスクトップをインストールしている場合、
   同期フォルダを出力先に指定することで可能です。

Q. 何号証まで対応していますか？
A. システム上は9999号証まで対応しています。

詳しくは「操作説明書.pdf」のFAQをご覧ください。

■ トラブルシューティング
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

【ツールが起動しない】
・Windows 11（64bit）であることを確認してください
・ウイルス対策ソフトでブロックされていないか確認してください

【エラーが表示される】
・PDFファイルが破損していないか確認してください
・ファイル名に特殊文字（¥/:*?"<>|）が含まれていないか確認してください
・出力先フォルダに書き込み権限があるか確認してください

【処理が遅い】
・PDFのページ数が多い場合、サムネイル生成に時間がかかります
・500ページを超える場合は、分割して処理することをおすすめします
